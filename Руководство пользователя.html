О MagicKeys


MagicKeys - это программа для обеспечения доступности музыкальных VST плагинов в секвенсоре Reaper. При помощи данного ПО вы можете взаимодействовать с элементами интерфейса, недоступными для программ чтения с экрана. Основное внимание уделяется доступности плагина Kontakt начиная с 6 версии и библиотек для него. Доступность других плагинов реализуется разработчиком по мере возможности.

Настройка системы перед использованием


Для корректной работы MagicKeys, перед использованием необходимо произвести следующие настройки:
  • Установите максимальное разрешение доступное для вашего основного монитора.
  • Установите параметры масштабирования изображения на экране на 100%.
  • В свойствах бинарного файла MagicKeys, на вкладке совместимость отметьте галочку "Запускать эту программу от имени администратора".
  • Убедитесь в стабильности вашего интернет соединения.
  • При необходимости, добавьте основной бинарный файл MagicKeys в список исключений вашего антивируса.
  • Запустите основной бинарный файл MagicKeys и следуйте дальнейшим подсказкам программы.

Системные требования


  • ОС Windows 10 не ниже 1603 x86-x64.
  • Установленный компонент .NET Core Runtime x86.
  • Установленная программа экранного доступа NVDA или jaws не ниже версии 16.0.
  • Установленный, или портативный Cockos Reaper не ниже 5 версии

Основные понятия и клавиши


И так, вы успешно всё установили, запустили, зарегистрировались, теперь можно приступать к самому главному.
Далее описаны глобальные сочетания клавиш. Их действие распространяется на всю ОС.

Глобальные горячие клавиши


  • Ctrl+Shift+F1 - Включить выключить MagicKeys.
  • Ctrl+Shift+F2 - Открыть диалог выхода из MagicKeys.
  • Ctrl+Shift+F3 - Открыть окно справки для текущего плагина.
  • Ctrl+Shift+F4 - Предотвращает зацикливание в некоторых ситуациях. Использовать только при полной уверенности в зависании.
  • Ctrl+Shift+F5 - включение/отключение звуковой схемы.


Запустите MagicKeys и reaper.
Создайте дорожку в reaper и добавьте на неё Native Instruments Kontakt.
Откройте окно плагина.
Как всё это сделать описано в документации к дополнению которое используется для доступа к reaper. Например osara.
Основной способ взаимодействия с VST плагинами в MagicKeys реализован в виде псевдо-интерфейса.
Дополнительные сочетания клавиш можно просмотреть вызвав клавиатурную справку по комбинации Ctrl+Shift+F3.

Клавиши управления псевдо-интерфейсом


  • Tab - Перемещение виртуального навигатора на объект вперёд.
  • Shift+Tab - Перемещение виртуального навигатора на объект назад.
  • Enter - Активация объекта под виртуальным навигатором.
  • F1 - Проговаривание краткой информации о текущем объекте навигатора.

так же существует альтернативный способ навигации по элементам виртуального интерфейса.
Для его активации перейдите в окно настроек MagicKeys через меню на панели задач и отметьте соответствующий пункт.
  • Right - Перемещение виртуального навигатора на объект вперёд.
  • Left - Перемещение виртуального навигатора на объект назад.
  • Enter - Активация объекта под виртуальным навигатором.
  • F1 - Проговаривание краткой информации о текущем объекте навигатора.

В некоторых плагинах активация элементов производится по нажатию клавиши space.
Это необходимо для взаимодействия по клавише Enter с элементами, доступными для программ экранного доступа.
Данная особенность описана в глобальной справке по текущему плагину. Глобальная справка доступна по клавишам Ctrl+Shift+F3.

Типы и особенности объектов виртуального интерфейса


Начиная с версии 1.4 объекты виртуального интерфейса в MagicKeys подразделяются на типы.
Управление проговариванием типов доступно в окне настроек.

Основные типы объектов


  • Button - Обычная кнопка. Может выполнять сразу несколько действий, открытие меню, клик по пунктам, и другие типы взаимодействия с элементами.
  • Slider - Позволяет управлять ползунками и другими их разновидностями. Предоставляет пользователю диалог для ввода значения, производит монипуляции для установки необходимого состояния элемента
  • Switch - Циклически переключает сгрупированные элементы RadioBox.
  • CheckBox - Управляет состоянием флажков, в некоторых случаях и другими объектами виртуального интерфейса.

Объекты виртуального интерфейса с типами CheckBox и Switch могут изменять состояние видимости других объектов.
Если после CheckBox следуют связанные с ним объекты, то при определённом состоянии CheckBox они могут быть скрыты.
Данная особенность может быть применима и к объекту Switch.
В MagicKeys виртуальные объекты располагаются в соответствии с порядком в окне. Слева направо, сверху вниз.
При командной работе для передачи указаний по управлению, вы можете использовать этот порядок для ориентира.
